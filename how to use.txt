вы можете загрузить треки локально, для этого рядом с файлом main.js в папке tracks, добавьте папку 1...2...3 и так далее, в ней должен находится сам трек и его обложка(.mp3 и .jpg форматы), пример есть в папке tracks/1


Установка необходимого ПО

Установите Node.js:

Скачайте Node.js с официального сайта

Установите его, следуя инструкциям установщика

Проверьте установку, выполнив в командной строке:


node -v
npm -v
Установите зависимости:

Откройте командную строку в папке с проектом

Выполните команду:


npm install
Настройка проекта
Добавьте Client ID:

Откройте файл main.js в текстовом редакторе

Найдите строку с CLIENT_ID: 'put ur client_id here'

Замените 'put ur client_id here' на ваш реальный Client ID (в кавычках)

Примечание: В настоящее время разрабатывается расширение для автоматического получения Client ID. Пока его нет, вам нужно вставить ID вручную.

Запуск проекта
Запустите приложение:

В командной строке выполните:

npm start
Или используйте команду, указанную в файле package.json в разделе scripts
